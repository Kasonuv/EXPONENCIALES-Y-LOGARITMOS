Pantalla 9 · Dominio

¿Por qué el logaritmo solo acepta positivos?

La función logarítmica de base positiva solo puede recibir números positivos.

No puedo calcular log2(-8), porque ninguna potencia de base positiva da como resultado -8.
Dominio de f(x)=logb(x):   x>0

Punto importante

La función pasa por (1,0), porque b0=1.

Pantalla 11 · Cierre

Síntesis final

Exponencial: describe qué ocurre cuando una cantidad se multiplica repetidamente.
Logaritmo: permite descubrir el exponente, es decir, cuántas veces ocurrió esa multiplicación.
by=x   ⟺   logb(x)=y

Para llevarte de esta lección

  • Si la base de una exponencial es mayor que 1, la función crece.
  • Si la base está entre 0 y 1, la función decrece.
  • El logaritmo es la operación inversa de la exponencial.
  • El logaritmo solo admite argumentos positivos.
